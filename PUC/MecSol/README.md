# Tema do projeto: 
Proposta de criação de um programa para solução de problemas de Estática e Resistência dos Materiais.

## Objetivo: 
Fornecer uma solução em cálculo estrutural (app, executável em ambiente web ou script) para análise de treliças simples (rígidas).

## Requisitos de projeto:
+ O usuário inicia sua análise fornecendo a localização no plano de todos os nós da treliça
+ Em seguida, ele fornece quais nós pertencem a um mesmo elemento e, consequentemente,
como os elementos estão conectados entre si
+ O programa deve avisar ao usuário que elementos estão se cruzando, fornecendo uma
mensagem de erro
+ Na sequência o usuário especifica quais são os vínculos que mantém a treliça equilibrada
(pino, rolete, etc.)
+ O usuário também deve fornecer a(s) força(s) (intensidade, direção e sentido ou suas
projeções) e em qual(is) nós ela(s) está aplicada
+ O programa deverá entregar todas as forças nos nós e nos vínculos. Deverá mostrar quais
elementos estão em tração ou compressão
+ Além disso, ao apresentar os resultados, o programa deve avisar ao usuário quais elementos
possuem força zero

# Estrutura de avaliação:
## Etapa 1: Levantamento sobre aplicativos existentes.
+ O que fazer? Pesquisar sobre o mercado de aplicativos que resolvem problemas de
treliças. Levantar idiomas, apontar as dificuldades e facilidades de uso
comparativamente às boas práticas de experiência de usuário. Documentar o que
foi encontrado por meio prints das telas
## Etapa 2: Implementação do método de cálculo de treliça.
+ O que fazer? Desenvolver um aplicativo/script onde o usuário possa inserir as
informações necessárias para a solução do seu problema de treliça (ver
Requisitos); Implementar os cálculos necessários para a solução do problema.
## Etapa 3: Etapa de teste com público-alvo; Aprimoramento
+ O que fazer? Submeter o aplicativo para avaliação de estudantes de outras
Engenharias (especialmente Mecânica e Civil).
